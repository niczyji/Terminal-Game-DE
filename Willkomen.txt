Willkommen!

Dies ist ein Spiel, das Ihnen die Grundlagen der Benutzung eines POSIX (Linux, BSD, UNIX)-Terminals vermittelt.

Um das Spiel zu beginnen, öffnen Sie ein Terminal.

Geben Sie die Buchstaben "cd " (nur die Buchstaben, nicht die Anführungszeichen) in das Terminal ein...

Dann ein Leerzeichen (drücken Sie die Leertaste)...

Ziehen Sie dann das Verzeichnis ``Terminal-Game`` aus diesem Ordner in Ihr Terminal.

Wenn Ihr Terminal Sie fragt, was Sie mit dem soeben hineingezogenen Verzeichnis tun wollen, wählen Sie "Ort einfügen".
Wenn es nicht fragt, dann erwarten Sie, dass es den Dateipfad des Ordners, den Sie gerade hineingezogen haben, einfügt.

Sobald Sie etwas wie

```
$ cd /home/Ihr_Benutzername /Downloads/Terminal-Game/Eingang
```

in Ihrem Terminal Fenster, drücken Sie RETURN(ENTER).
Der genaue Pfad zum Eingang variiert, je nachdem, wo Sie die Datei gespeichert haben.

Ihr erster Schritt ist sehr wichtig.
Geben Sie dies in Ihr Terminal ein:

```
cd Eingang
cat Papierrolle
```

Sie spielen jetzt das Spiel.
